## dsUI01

学习了基本框架

主窗口：root = tk.TK()，需要有title和geometry（宽高）

文本：label，要有文本和字体大小

按键：button，要有command（执行的函数）

输入框：entry，其内容用entry.get来得到

消息框，在事件中打开，使用box.showinfo()，里面直接输入标题文本即可

最后用mainloop()来保持窗口

整个创建是按照从上到下的顺序，通过pack函数来决定和上一个框的间距
